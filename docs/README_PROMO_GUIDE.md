## 🚀 快速使用指南

### 1. 更新项目根目录的 README

```bash
# 在项目根目录执行：
cp docs/README_CN.md README.md      # 用高转化中文版替换现有 README
# 或：手动把 README_CN.md 中你觉得合适的段落拷贝进 README.md
```

**为什么这样做**：当前 README 偏向"项目文档"风格；新写的 `README_CN.md` 是"用户转化"风格：
- 开头有清晰的「你将获得什么 / 用前 vs 用后」对比表
- 徽章、CTA 多次出现（Star 按钮在顶部和底部）
- 一句话定位反复强化：`不是监控工具 — 是 AI 的进化引擎`
- 架构图 + 路线图透明化未来方向，让潜在贡献者能快速入坑

- 抖音：视频简介 / 置顶评论 / 合集名称里
- 知乎 / V2EX / 掘金：直接贴链接

- 省了 30% Token
